# Websocket Bridge for Executors

Allows external executions via a websocket, meant for external UI, or for clients that want a remotely execute code without using their executor.
Requires WebSocket sUNC support.

Server should be on the same machine as the executor --> Run run.bat

Client includes a default electron-js executor, website executor, and the code for the executor to run

Place the Executor code in your Auto Execute folder and adjust the address and port accordingly if needed
